## Om detta repository

Detta repository innehåller nätverkssimuleringsprojekt och laborationsövningar utvecklade med Cisco Packet Tracer. Projektet fokuserar på att lära sig nätverksarkitektur, konfiguration och implementering av både trådbundna och trådlösa nätverk.

##  Innehåll

Repositoryt innehåller följande Cisco Packet Tracer-filer (.pka):

### 4.6.5 Packet Tracer - Connect a Wired and Wireless LAN
En omfattande övning som demonstrerar hur man kopplar samman trådbundna och trådlösa lokala nätverk (LAN). Detta projekt täcker:
- Konfiguration av trådbundna nätverksanslutningar
- Implementering av trådlösa åtkomstpunkter
- Integration mellan olika nätverkstyper
- Grundläggande nätverksarkitektur

### Lab3_student.pka
Tredje laborationsövningen som innehåller praktiska nätverkskonfigurationsuppgifter. Denna laboration fokuserar på:
- Avancerad nätverkskonfiguration
- Felsökning av nätverksproblem
- Praktisk tillämpning av nätverksprinciper

### Network 1.pka
Ett grundläggande nätverksprojekt som demonstrerar fundamentala nätverkskoncept och konfigurationer.

## Teknologi & Verktyg

- **Plattform:** Cisco Packet Tracer
- **Fokusområden:** 
  - Nätverksdesign och -arkitektur
  - Router- och switch-konfiguration
  - Trådbundna och trådlösa LAN
  - TCP/IP-protokoll
  - Nätverksfelsökning

## Komma igång

### Förutsättningar

För att öppna och arbeta med filerna i detta repository behöver du:

- **Cisco Packet Tracer** (senaste versionen rekommenderas)
  - Ladda ner från: [Cisco Networking Academy](https://www.netacad.com/courses/packet-tracer)
  - Gratis för studenter och lärare
  - Tillgänglig för Windows, macOS och Linux

### Installation

1. Klona repositoryt:
```bash
git clone https://github.com/IbrahimAwad98/Communication-Networks.git
cd Communication-Networks
```

2. Öppna .pka-filerna i Cisco Packet Tracer:
```bash
# Dubbelklicka på filen eller öppna via Packet Tracer
# File -> Open -> Välj önskad .pka-fil
```

## Lärandemål

Detta repository täcker följande nätverkskoncept:

- **Grundläggande nätverk:**
  - IP-adressering och subnetting
  - OSI-modellen och TCP/IP
  - Nätverksenheter (routers, switches, access points)

- **LAN-konfiguration:**
  - Ethernet och kabelbaserade nätverk
  - VLAN-implementering
  - Switch-konfiguration

- **Trådlösa nätverk:**
  - WLAN-setup och konfiguration
  - Säkerhet för trådlösa nätverk
  - Integration med trådbundna nätverk

- **Felsökning:**
  - Ping och traceroute
  - Diagnostikverktyg
  - Problemlösning i nätverk

## Användning

Varje .pka-fil innehåller:
- Förkonfigurerade nätverkstopologier
- Instruktioner och uppgifter
- Testscenarier för verifiering

**Rekommenderat arbetsflöde:**
1. Öppna filen i Cisco Packet Tracer
2. Läs instruktionerna (vanligtvis under "Instructions" i PT)
3. Konfigurera nätverksenheterna enligt uppgiften
4. Testa konfigurationen med ping och andra verktyg
5. Verifiera att alla krav uppfylls
